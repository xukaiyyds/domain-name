## 域名证书

一个简约的网页版顶级国际域名证书，方便自己或他人随时查看你的域名剩余时长以及域名详细信息。

### 使用

直接克隆本仓库，然后将`index.html`的域名信息改为你自己的即可（在下面的`<script></script>`部分改）。

```bash
git clone git@github.com:xukaiyyds/domain-name.git
```
