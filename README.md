## domain-name

一个简约的网页版顶级国际域名证书，方便自己或他人查看你的域名使用时长以及域名剩余时长。

### 使用

直接`Fork`本仓库，然后将首页`index.html`中的`<script></script>`部分改为你自己的域名信息即可。

### 特征

- [x] 响应式设计
- [x] 移动端适配
- [x] 个性化配置

### 功能

- [x] 浅色主题
- [x] 操作反馈
- [x] 分享证书

功能部分对应用到了以下插件：

- [Darkmode.js](https://github.com/sandoche/Darkmode.js)
