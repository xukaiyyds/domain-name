## 域名证书

一个简约的高仿版顶级国际域名证书，方便自己或他人随时查看你的域名到期时间以及详细域名信息。

### 使用

直接克隆本仓库，然后将`index.html`的域名信息改为你自己的即可（在下面的`<script></script>`部分改）。

```bash
git clone git@github.com:xukaiyyds/domain-name.git
```

### 特征

- 响应式设计 ✅
- 精确计算域名使用时间和域名续费倒计时 ✅
- 根据域名续费倒计时显示不同的颜色提醒 ✅

### 功能

- 深色模式 ✅
- 中英文翻译
- 个性化设置
  - 是否禁用深色模式
  - 是否取消背景图片
